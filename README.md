## Run Locally

Clone the project

```bash
  git clone https://github.com/Kavita539/pomodro-app.git
```

Go to the project directory

```bash
  cd kayyvisuals-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Features Available.

- Home page with category section
- Video listing page with category filter.
- Playlist Management
- Like/Dislike
- Watch Later
- History
- Authentication
  - Login
  - Sigup
  - Guest Login
- Fully Resposive
- Single Video Page
- Loading & Alerts
